# Gunjan Sarkar — Portfolio

Personal portfolio website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Development

```sh
npm install
npm run dev
```

## Deployment

Build for production:

```sh
npm run build
```
